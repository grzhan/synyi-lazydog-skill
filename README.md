# Synyi Lazy Dog Skill

## 功能

- **录入特定日期工时**：指定某一天，快速录入当天的工时。
- **自动匹配补录流程录入当月工时**：自动识别补录流程，一键完成当月工时的录入。

## 指令样例

录入特定日期工时：

```
帮我录入 2026-05-26 的工时，项目编号 Y3018005，工作内容 Nexus 联调
```

自动匹配补录流程录入当月工时：

```
基于补录流程把这个月的工时都补一下，项目用 Y3018005，工作内容写「Nexus 开发」
```

结合龙虾使用效果更佳。
